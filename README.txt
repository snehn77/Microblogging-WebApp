To Run this Application there are to steps 

1)To Run Backend

-> Open The Sneh.Microblogging.Application.sln in visual studio
-> Rebuild the solution
-> Clean Solution
-> Build the Solution
-> Run the Backend you see a browser pop up thats means the backend is started


2) To run FrontEnd

-> Open SnehMicroblog.UI/client path
-> Open this client folder vscode 
-> Once check if node_modules folder is present if not present run npm i if present no need
-> Hit ng serve -o command and you see our app opening in the browser
